Connectors provide a unified interface to trade across centralized exchanges (CEX), decentralized exchanges (DEX), and blockchain networks.

## Connector Types

### Spot Connectors

Trade on spot markets across centralized exchanges.

| Exchange | Connector Name |
|----------|---------------|
| Binance | `binance` |
| Coinbase | `coinbase_pro` |
| Kraken | `kraken` |
| KuCoin | `kucoin` |
| OKX | `okx` |

**Configuration**:

```bash
curl -X POST http://localhost:8000/accounts/add-credential/master_account/binance \
  -u admin:admin \
  -H "Content-Type: application/json" \
  -d '{
    "binance_api_key": "your-api-key",
    "binance_api_secret": "your-api-secret"
  }'
```

### Perpetual Connectors

Trade perpetual futures with leverage.

| Exchange | Connector Name |
|----------|---------------|
| Binance Perpetual | `binance_perpetual` |
| Hyperliquid | `hyperliquid_perpetual` |
| Bybit | `bybit_perpetual` |
| OKX | `okx_perpetual` |

**Position Modes**:

| Mode | Description |
|------|-------------|
| `ONEWAY` | Single position per trading pair; BUY/SELL orders net against each other |
| `HEDGE` | Separate long and short positions tracked independently |

**Configuration**:

```bash
curl -X POST http://localhost:8000/accounts/add-credential/master_account/hyperliquid_perpetual \
  -u admin:admin \
  -H "Content-Type: application/json" \
  -d '{
    "hyperliquid_api_key": "0x1234...",
    "hyperliquid_api_secret": "your-secret-key"
  }'
```

### Network Connectors

Trade on DEXs and blockchain networks via Hummingbot Gateway.

| Network | DEX Protocols |
|---------|--------------|
| Solana | Jupiter, Raydium, Orca, Meteora |
| Ethereum | Uniswap |
| Base | Uniswap |

**Requirements**:

- Gateway instance deployed
- Wallet configured for the network
- RPC endpoint configured

**Configuration via Telegram**:

1. `/config` → **Gateway** to deploy and configure Gateway
2. `/config` → **Wallets** to create or import wallets
3. `/config` → **Networks** to configure RPC endpoints

## List Available Connectors

```bash
curl -u admin:admin http://localhost:8000/connectors
```

## Check Connector Configuration

Get required credential fields:

```bash
curl -u admin:admin http://localhost:8000/connectors/hyperliquid/config-map
```

## Get Trading Rules

Check order constraints before trading:

```bash
curl -u admin:admin "http://localhost:8000/connectors/hyperliquid/trading-rules?trading_pairs=HYPE-USDC"
```

Response:

```json
{
  "HYPE-USDC": {
    "min_order_size": 0.1,
    "min_price_increment": 0.01,
    "min_base_amount_increment": 0.001,
    "supports_market_orders": true
  }
}
```

<Note>
Spot and perpetual markets are separate connectors. For example, `binance` (spot) and `binance_perpetual` (perps) are different connectors with different credential configurations.
</Note>
