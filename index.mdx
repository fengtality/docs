Welcome to the documentation for **Condor**, the Telegram interface for Hummingbot, and the **Hummingbot API** backend that powers it.

## What is Condor?

Condor is a powerful Telegram bot that provides a mobile-friendly interface to interact with your Hummingbot trading infrastructure. Monitor portfolios, manage bots, execute trades on CEX and DEX exchanges, and configure your entire trading setup—all from Telegram.

## What is Hummingbot API?

The Hummingbot API is a comprehensive RESTful API framework for managing trading operations across multiple exchanges. It serves as the backend for Condor and provides direct API access for developers.

## Quick Start

<Steps>
  <Step title="Install Hummingbot API">
    ```bash
    git clone https://github.com/hummingbot/hummingbot-api.git
    cd hummingbot-api
    make setup && make deploy
    ```
  </Step>
  <Step title="Install Condor">
    ```bash
    git clone https://github.com/hummingbot/condor.git
    cd condor
    ./setup-environment.sh
    docker compose up -d
    ```
  </Step>
  <Step title="Connect via Telegram">
    Open your bot in Telegram, send `/start`, then use `/config` to connect to your Hummingbot API instance.
  </Step>
</Steps>

## Architecture

```
Telegram User
     ↓
Condor Bot ──→ Hummingbot API ──→ Exchanges (CEX)
     │              │
     │              └──→ Trading Bots
     │
     └──→ Gateway ──→ DEX Protocols
```
