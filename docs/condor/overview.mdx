Condor is a Telegram bot that provides mobile access to your Hummingbot trading infrastructure. Monitor portfolios, manage bots, and execute trades from anywhere.

## Features

| Feature | Description |
|---------|-------------|
| Portfolio Dashboard | Real-time tracking with PnL indicators, holdings breakdown, and graphical analysis |
| Bot Management | Monitor status, view logs, edit parameters, and control trading bots |
| CEX Trading | Place orders on centralized exchanges with leverage configuration |
| DEX Swaps | Execute token swaps via Gateway on Solana, Ethereum, Base, and more |
| Liquidity Management | Manage CLMM positions, collect fees, and monitor pools |
| Configuration | Manage API servers, exchange credentials, and Gateway settings |

## Architecture

Condor connects to your Hummingbot API server to access trading functionality:

```
┌─────────────────┐
│     Condor      │
│   (Telegram)    │
└────────┬────────┘
         │
         ▼
┌─────────────────────────────────────────────┐
│           Hummingbot API Server             │
│  ┌─────────────┐  ┌─────────────────────┐   │
│  │  Portfolio  │  │  Bot Orchestration  │   │
│  │  Tracking   │  │     via EMQX        │   │
│  └─────────────┘  └─────────────────────┘   │
└─────────────────────┬───────────────────────┘
                      │
      ┌───────────────┼───────────────┐
      ▼               ▼               ▼
┌───────────┐   ┌───────────┐   ┌───────────┐
│  Gateway  │   │    CEX    │   │   Bots    │
│   (DEX)   │   │ Exchanges │   │ Instances │
└───────────┘   └───────────┘   └───────────┘
```
