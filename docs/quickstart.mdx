This guide walks you through deploying Hummingbot API on a cloud server and controlling it with Condor, a Telegram bot interface.

## Prerequisites

| Component | Requirement |
|-----------|-------------|
| **OS** | Linux x64/ARM (Ubuntu 20.04+) |
| **RAM** | 4 GB minimum |
| **Storage** | 10 GB SSD |
| **CPU** | 2+ vCPUs |

<Tip>
Recommended cloud providers: AWS, Google Cloud, or DigitalOcean
</Tip>

## 1. Set Up a Cloud Server

Create a Linux server with your preferred cloud provider. For AWS:

1. Launch an EC2 instance with Ubuntu 22.04 LTS
2. Select `t3.medium` or larger instance type
3. Configure security group to allow ports 22 (SSH) and 8000 (API)

## 2. Install Docker

Connect to your server via SSH and install Docker:

```bash
curl -fsSL https://get.docker.com -o get-docker.sh
sh get-docker.sh
```

## 3. Deploy Hummingbot API

Clone the repository and run the setup:

```bash
git clone https://github.com/hummingbot/hummingbot-api.git
cd hummingbot-api
chmod +x setup.sh
./setup.sh
make deploy
```

The API server will start on port 8000. Access the Swagger UI at `http://your-server-ip:8000/docs`.

## 4. Set Up Condor Telegram

### Create a Telegram Bot

1. Open Telegram and search for [@BotFather](https://t.me/botfather)
2. Send `/newbot` and follow the prompts
3. Copy the bot token (format: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Get Your Telegram User ID

1. Search for [@userinfobot](https://t.me/userinfobot) on Telegram
2. Send `/start` to receive your user ID

### Install Condor

```bash
git clone https://github.com/hummingbot/condor.git
cd condor
./setup-environment.sh
```

When prompted, enter:
- Your Telegram bot token
- Your authorized user IDs (comma-separated)

### Launch Condor

```bash
docker compose up -d
```

## 5. Connect to Your API

1. Open your Telegram bot
2. Send `/start` to begin
3. Go to `/config` → **API Servers**
4. Add your Hummingbot API server:
   - Host: your server IP address
   - Port: 8000
   - Username: admin
   - Password: admin

## 6. Add Exchange Credentials

1. In Condor, go to `/config` → **API Keys**
2. Select an exchange (e.g., Binance, Hyperliquid)
3. Enter your API key and secret

## 7. Start Trading

You can now use Condor to:
- `/portfolio` - View balances across all exchanges
- `/trade` - Place orders on CEX markets
- `/bots` - Deploy and manage trading bots
- `/swap` - Execute DEX swaps (requires Gateway)

See [Condor Commands](/docs/condor/commands) for the full command reference.
