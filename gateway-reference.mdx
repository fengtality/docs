Gateway provides a unified API for interacting with decentralized exchanges across multiple blockchain networks. This reference documents Gateway's native API endpoints.

## Base URL

```
http://localhost:15888
```

When accessed through Hummingbot API proxy:

```
http://localhost:8000/gateway-proxy
```

## API Categories

| Category | Description |
|----------|-------------|
| [Config](/gateway-reference/config/get-config) | Gateway configuration and settings |
| [Wallet](/gateway-reference/wallet/get-wallet) | Wallet management for DEX trading |
| [Tokens](/gateway-reference/tokens/get-tokens) | Token registry and management |
| [Pools](/gateway-reference/pools/get-pools-trading-pair) | AMM pool discovery and management |
| [Trading - Swap](/gateway-reference/trading-swap/get-trading-swap-quote) | Token swap quotes and execution |
| [Trading - CLMM](/gateway-reference/trading-clmm/get-trading-clmm-pool-info) | Concentrated liquidity position management |
| [Chains](/gateway-reference/chains-ethereum/get-chains-ethereum-status) | Blockchain-specific operations |

## Interactive Playground

Each endpoint page includes an interactive playground where you can test API calls directly.

<Note>
The playground makes requests directly to your Gateway server. Ensure Gateway is running and accessible at the configured URL.
</Note>

## Response Format

**Success responses** return the requested data:

```json
{
  "network": "mainnet",
  "timestamp": 1234567890,
  "latency": 0.5
}
```

**Error responses** include error details:

```json
{
  "error": "Token not found",
  "code": "TOKEN_NOT_FOUND"
}
```
