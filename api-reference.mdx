The Hummingbot API is a comprehensive RESTful API for managing trading operations across centralized and decentralized exchanges. This reference documents all available endpoints.

## Base URL

```
http://localhost:8000
```

For remote deployments, replace `localhost` with your server's address.

## Authentication

All endpoints require HTTP Basic Authentication:

```bash
curl -u admin:admin http://localhost:8000/accounts
```

Or with the Authorization header:

```bash
curl -H "Authorization: Basic $(echo -n 'admin:admin' | base64)" http://localhost:8000/accounts
```

## Interactive Playground

Each endpoint page includes an interactive playground where you can test API calls directly. Enter your credentials and server URL to make live requests.

<Note>
The playground makes requests directly to your API server. Ensure your Hummingbot API is running and accessible.
</Note>

## Response Format

**Success responses** return the requested data:

```json
["master_account"]
```

**Error responses** include a `detail` field:

```json
{
  "detail": "Account not found"
}
```
