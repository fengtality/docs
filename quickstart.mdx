This guide walks you through setting up Condor, connecting to Telegram, adding exchange credentials, and using the web dashboard and agent modes.

## 1. Install Condor

Clone the repository and run the install script:

```bash
git clone https://github.com/hummingbot/condor.git
cd condor

make install     # Interactive setup + dependencies + AI CLI tools
make run         # Start the bot
```

See [Installation](/installation) for system requirements and cloud server options.

## 2. Connect Telegram

Condor uses Telegram as its primary interface. You'll need to create a bot and get your user ID.

### Create a Telegram Bot

1. Open [@BotFather](https://t.me/botfather) in Telegram
2. Send `/newbot` and follow the prompts to name your bot
3. Copy the bot token (format: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Get Your User ID

1. Open [@userinfobot](https://t.me/userinfobot) in Telegram
2. Send `/start` to receive your user ID

### Configure Condor

During installation, you'll be prompted to enter:
- **Bot Token**: The token from BotFather
- **User IDs**: Your Telegram user ID (comma-separated for multiple users)

Once configured, open your bot in Telegram and send `/start` to see the main menu.

## 3. Add Exchange Keys

Connect your exchange accounts to start trading.

1. In Telegram, send `/keys`
2. Select **Perpetual** or **Spot**
3. Choose an exchange (e.g., Binance, Hyperliquid)
4. Enter your API key and secret

<Warning>
For security, only enable **read + trade** permissions on your API keys. Never enable withdraw or transfer permissions.
</Warning>

### Verify Connection

After adding keys, check your balances:

```
You: What's my balance on Binance?
Condor: Here's your Binance balance:
- USDT: 1,234.56
- BTC: 0.025
```

Or use `/portfolio` to see all connected accounts.

## 4. Web Dashboard

Condor includes a web dashboard for visual monitoring and control.

### Launch the Dashboard

In Telegram, send `/web` to get a link to your dashboard. The link includes an encrypted token tied to your Telegram session—no separate login required.

### Dashboard Features

- **Portfolio**: View balances and positions across all exchanges
- **Executors**: Monitor and manage active executors
- **Agents**: View Trading Agent sessions and snapshots
- **Deploy**: Create grids and other executors visually

## 5. Agent Mode

Condor has two interaction modes: **Condor** (direct commands) and **Agent Builder** (create Trading Agents).

### Condor Mode (Default)

Talk to Condor directly using natural language or commands:

```
You: Analyze BTC-USDT and tell me what you think
Condor: Running technical analysis...
- Trend: Bullish (EMAs aligned)
- Support: $62,500
- Resistance: $65,200
- Suggestion: Consider grid with long bias
```

```
You: Deploy a grid on SOL-USDT with $100
Condor: Grid deployed on Binance SOL-USDT
- Amount: $100
- Range: $140 - $160
- Orders: 10
```

### Agent Builder Mode

Create autonomous Trading Agents that run on their own:

1. Send `/agent` in Telegram
2. Select **Agent Builder**
3. Follow the guided flow:
   - **Phase 1**: Strategy design (what you want to trade)
   - **Phase 2**: Market data (indicators, routines)
   - **Phase 3**: Strategy logic (agent.md)
   - **Phase 4**: Dry run (test reasoning)
   - **Phase 5**: Run once (test with real money)
   - **Phase 6**: Deploy live (loop mode)

Once deployed, the agent runs autonomously—observing markets, reasoning about strategy, and executing trades at each tick.
