Condor Dashboard is a modern, browser-based interface for managing trading bots and agents. It syncs with Condor Telegram for seamless mobile-to-desktop workflows.

<Info>
The web dashboard is under active development. Follow [#condor-feedback](https://discord.gg/hummingbot) for updates.
</Info>

## Features

<CardGroup cols={2}>
  <Card title="Trade" icon="chart-line">
    View market data (order book, price charts), place orders, and monitor positions

  <Card title="AMM Markets" icon="coins">

## Prerequisites

- Node.js 20.19+ or 22.12+
- A running Hummingbot API server

## Session Continuity

The web dashboard shares session state with Telegram:

- Start a Trading Agent on Telegram
- Monitor and adjust in the web dashboard
- Receive notifications on mobile

Same session, same agent state, same conversation history.

## Development

```bash
# Clone the dashboard repo
git clone https://github.com/hummingbot/dashboard.git
cd dashboard

# Install dependencies
npm install

# Start development server
npm run dev
```

The dashboard connects to your Hummingbot API server at `http://localhost:8000` by default.
