**Scripts** are single-file Python programs that run inside Hummingbot. They provide direct access to exchange connectors for simpler automation tasks.

## Scripts vs Controllers

| Aspect | Scripts | Controllers |
|--------|---------|-------------|
| **Complexity** | Simple, single-file | Modular, multi-component |
| **Use Case** | Custom indicators, alerts, simple strategies | Market making, complex strategies |
| **Executor Integration** | Manual | Built-in orchestration |
| **Learning Curve** | Lower | Higher |

## Script Structure

```python
from hummingbot.strategy.script_strategy_base import ScriptStrategyBase

class MyScript(ScriptStrategyBase):
    # Define markets to connect
    markets = {"binance": {"BTC-USDT", "ETH-USDT"}}

    def on_tick(self):
        """Called every tick (default: 1 second)."""
        # Your trading logic here
        pass
```

## Example: Price Alert

```python
from hummingbot.strategy.script_strategy_base import ScriptStrategyBase
from decimal import Decimal

class PriceAlertScript(ScriptStrategyBase):
    markets = {"binance": {"BTC-USDT"}}

    # Configuration
    alert_price = Decimal("70000")
    alerted = False

    def on_tick(self):
        price = self.connectors["binance"].get_mid_price("BTC-USDT")

        if price > self.alert_price and not self.alerted:
            self.notify(f"BTC crossed ${self.alert_price}! Current: ${price}")
            self.alerted = True
        elif price < self.alert_price:
            self.alerted = False  # Reset for next crossing
```

## Example: Simple DCA

```python
from hummingbot.strategy.script_strategy_base import ScriptStrategyBase
from decimal import Decimal
import time

class DCAScript(ScriptStrategyBase):
    markets = {"binance": {"BTC-USDT"}}

    # Configuration
    buy_amount = Decimal("0.001")
    interval_seconds = 3600  # 1 hour
    last_buy_time = 0

    def on_tick(self):
        current_time = time.time()

        if current_time - self.last_buy_time >= self.interval_seconds:
            self.buy(
                connector_name="binance",
                trading_pair="BTC-USDT",
                amount=self.buy_amount,
                order_type=OrderType.MARKET
            )
            self.last_buy_time = current_time
            self.notify(f"DCA buy: {self.buy_amount} BTC")
```

## Example: Spread Monitor

```python
from hummingbot.strategy.script_strategy_base import ScriptStrategyBase
from decimal import Decimal

class SpreadMonitorScript(ScriptStrategyBase):
    markets = {
        "binance": {"BTC-USDT"},
        "coinbase_pro": {"BTC-USD"}
    }

    spread_threshold = Decimal("0.005")  # 0.5%

    def on_tick(self):
        binance_price = self.connectors["binance"].get_mid_price("BTC-USDT")
        coinbase_price = self.connectors["coinbase_pro"].get_mid_price("BTC-USD")

        spread = abs(binance_price - coinbase_price) / coinbase_price

        if spread > self.spread_threshold:
            self.notify(
                f"Spread alert: {spread:.2%}\n"
                f"Binance: ${binance_price}\n"
                f"Coinbase: ${coinbase_price}"
            )
```

## Script Methods

| Method | Description |
|--------|-------------|
| `on_tick()` | Called every tick interval |
| `buy()` | Place a buy order |
| `sell()` | Place a sell order |
| `cancel()` | Cancel an order |
| `notify()` | Send notification |
| `log_with_clock()` | Log with timestamp |

## Accessing Market Data

```python
def on_tick(self):
    connector = self.connectors["binance"]

    # Prices
    mid_price = connector.get_mid_price("BTC-USDT")
    bid_price = connector.get_price("BTC-USDT", is_buy=True)
    ask_price = connector.get_price("BTC-USDT", is_buy=False)

    # Order book
    order_book = connector.get_order_book("BTC-USDT")

    # Balances
    balance = connector.get_balance("USDT")
    available = connector.get_available_balance("USDT")
```

## Deploying a Script

**Via Telegram**:
1. `/bots` → Create New Bot
2. Select **Script**
3. Choose the script
4. Start the bot

**Via API**:

```bash
curl -u admin:admin -X POST http://localhost:8000/bot-orchestration/deploy \
  -H "Content-Type: application/json" \
  -d '{
    "bot_name": "price-alerts",
    "script_name": "price_alert_script",
    "config": {}
  }'
```
