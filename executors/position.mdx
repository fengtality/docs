The **Position Executor** manages perp and spot positions with configurable exit conditions using the Triple Barrier method.

## Properties

| Property | Value |
|----------|-------|
| Position Type | Perp or Spot (determined by connector) |
| keep_position | Controlled via stop action |
| P&L Calculation | Entry vs exit price ± fees ± funding |

## Configuration

```python
from hummingbot.strategy_v2.executors.position_executor.data_types import PositionExecutorConfig

position_config = PositionExecutorConfig(
    controller_id="my-agent",
    connector_name="binance_perpetual",
    trading_pair="SOL-USDT",
    side=TradeType.BUY,
    amount=Decimal("10.0"),
    entry_price=Decimal("150.0"),  # Optional limit entry
    leverage=5,
    triple_barrier_config=TripleBarrierConfig(
        take_profit=Decimal("0.02"),     # 2% take profit
        stop_loss=Decimal("0.01"),       # 1% stop loss
        time_limit=3600,                 # 1 hour max
        trailing_stop=TrailingStop(
            activation_price=Decimal("0.01"),
            trailing_delta=Decimal("0.005"),
        ),
    ),
)
```

## Triple Barrier Config

| Parameter | Type | Description |
|-----------|------|-------------|
| `take_profit` | Decimal | Percentage gain to close (e.g., `0.02` = 2%) |
| `stop_loss` | Decimal | Percentage loss to close (e.g., `0.01` = 1%) |
| `time_limit` | int | Maximum duration in seconds |
| `trailing_stop` | TrailingStop | Optional trailing stop configuration |

## Trailing Stop

| Parameter | Description |
|-----------|-------------|
| `activation_price` | Profit percentage to activate trailing stop |
| `trailing_delta` | Distance from peak to trigger stop |

## Exit Conditions

The position closes when any barrier is hit:

1. **Take Profit**: Price reaches take profit target
2. **Stop Loss**: Price reaches stop loss limit
3. **Time Limit**: Maximum duration exceeded
4. **Trailing Stop**: Trailing stop triggered after activation
5. **Early Stop**: Manual termination

## Use Cases

- Directional trades with defined risk
- Momentum strategies
- Hedging positions
- Scalping with tight stops
