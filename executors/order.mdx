---
title: "Order Executor"
description: "Limit and market orders on CEX or DEX"
---

The **Order Executor** places limit and market orders on centralized or decentralized exchanges.

## Properties

| Property | Value |
|----------|-------|
| Position Type | Spot |
| keep_position | `true` (default) |
| P&L Calculation | None - just order fill |

## Configuration

```python
from hummingbot.strategy_v2.executors.order_executor.data_types import OrderExecutorConfig

order_config = OrderExecutorConfig(
    controller_id="my-agent",
    connector_name="binance",
    trading_pair="BTC-USDT",
    side=TradeType.BUY,
    amount=Decimal("0.01"),
    position_action=PositionAction.OPEN,
    execution_strategy=ExecutionStrategy(
        order_type=OrderType.LIMIT,
        price=Decimal("65000.0"),
    ),
)
```

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `controller_id` | string | Agent ID for attribution |
| `connector_name` | string | Exchange connector |
| `trading_pair` | string | Market (e.g., `BTC-USDT`) |
| `side` | TradeType | `BUY` or `SELL` |
| `amount` | Decimal | Order size in base asset |
| `execution_strategy` | ExecutionStrategy | Order type and price |

## Use Cases

- Limit order entries at specific prices
- DCA purchases over time
- Building positions gradually
- Market orders for immediate execution
