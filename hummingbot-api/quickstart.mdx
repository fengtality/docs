This guide demonstrates how to use the Hummingbot API directly using `curl` or the Python client. It covers adding exchange credentials, viewing your portfolio, and placing orders.

## Prerequisites

- Hummingbot API installed and running (see [Installation Guide](/hummingbot-api/installation))
- Exchange API keys (see your exchange's documentation for creating API keys)
- Python 3.7+ with `hummingbot-api-client` installed (optional)

## Setup Python Client (Optional)

If you want to use the Python client for the examples below:

<Steps>
  <Step title="Install the client">
    ```bash
    pip install hummingbot-api-client
    ```
  </Step>
  <Step title="Create a new Python file">
    ```bash

        username="admin",
        password="admin"
    )
    ```
  </Step>
  <Step title="Run examples">
    ```bash
    python hummingbot_api_demo.py
    ```
  </Step>
</Steps>

## List Available Exchanges

Get a list of all available exchange connectors. Note that spot and perpetual markets are separate connectors (e.g., `hyperliquid` for spot and `hyperliquid_perpetual` for perps).

<Tabs>
  <Tab title="curl">
    ```bash
    curl -u admin:admin -X 'GET' \
      'http://localhost:8000/connectors/' \
      -H 'accept: application/json'
    ```

    **Response:**
    ```json
    [
      "binance",
      "binance_perpetual",
      "hyperliquid",
      "hyperliquid_perpetual",
      "okx",
      "okx_perpetual"
    ]
    ```
  </Tab>
  <Tab title="Python">
    ```python
    import asyncio
    from hummingbot_api_client import HummingbotAPIClient

    client = HummingbotAPIClient(
        base_url="http://localhost:8000",
        username="admin",
        password="admin"
    )

    async def list_exchanges():
        await client.init()
        try:
            connectors = await client.connectors.list_connectors()
            print("Available exchanges/connectors:")
            for connector in connectors:
                print(f"  - {connector}")
            return connectors
        finally:
            await client.close()

    asyncio.run(list_exchanges())
    ```
  </Tab>
</Tabs>

## Get Connector Configuration

Before adding credentials, check what configuration fields are required for your connector:

<Tabs>
  <Tab title="curl">
    ```bash
    curl -u admin:admin -X 'GET' \
      'http://localhost:8000/connectors/hyperliquid/config-map' \
      -H 'accept: application/json'
    ```

    **Response:**
    ```json
    [
      "hyperliquid_api_secret",
      "use_vault",
      "hyperliquid_api_key"
    ]
    ```
  </Tab>
  <Tab title="Python">
    ```python
    async def get_connector_config():
        await client.init()
        try:
            config_fields = await client.connectors.get_config_map("hyperliquid")
            print("Required configuration fields for hyperliquid:")
            for field in config_fields:
                print(f"  - {field}")
            return config_fields
        finally:
            await client.close()

    asyncio.run(get_connector_config())
    ```
  </Tab>
</Tabs>

## Add Exchange Credentials

Add your exchange credentials to the API. By default, only the `master_account` is created. You can add multiple accounts with different names if needed.

For Hyperliquid:
- `hyperliquid_api_secret`: Your Hyperliquid public address or vault address
- `hyperliquid_api_key`: Your API private key
- `use_vault`: Set to `true` if using vault address, `false` for normal account

<Tabs>
  <Tab title="curl">
    ```bash
    curl -X POST http://localhost:8000/accounts/add-credential/master_account/hyperliquid \
      -u "admin:admin" \
      -H "Content-Type: application/json" \
      -d '{
        "hyperliquid_api_key": "0x1234...abcd",
        "hyperliquid_api_secret": "your-private-key",
        "use_vault": false
      }'
    ```

    **Response:**
    ```json
    {
      "message": "Connector credentials added successfully"
    }
    ```
  </Tab>
  <Tab title="Python">
    ```python
    async def add_hyperliquid_credentials():
        await client.init()
        try:
            data = {
                "hyperliquid_api_key": "0x1234....abcde",
                "hyperliquid_api_secret": "your-api-private-key",
                "use_vault": False
            }
            result = await client.accounts.add_credential(
                "master_account", "hyperliquid", data
            )
            print("Added hyperliquid credentials:")
            print(result)
            return result
        finally:
            await client.close()

    asyncio.run(add_hyperliquid_credentials())
    ```
  </Tab>
</Tabs>

## View Your Portfolio

Check your portfolio balances across all connected exchanges:

<Tabs>
  <Tab title="curl">
    ```bash
    curl -u admin:admin -X 'POST' \
      'http://localhost:8000/portfolio/state' \
      -H 'accept: application/json' \
      -H 'Content-Type: application/json' \
      -d '{}'
    ```

    **Response:**
    ```json
    {
      "master_account": {
        "hyperliquid": [
          {
            "token": "USDC",
            "units": 100.00,
            "price": 1,
            "value": 100.00,
            "available_units": 100.00
          }
        ]
      }
    }
    ```
  </Tab>
  <Tab title="Python">
    ```python
    async def view_portfolio():
        await client.init()
        try:
            portfolio = await client.portfolio.get_state()
            print("\nPortfolio Summary")
            for account_name, exchanges in portfolio.items():
                print(f"\n{account_name}:")
                for exchange, balances in exchanges.items():
                    print(f"  {exchange}:")
                    for balance in balances:
                        print(f"    {balance['token']}:")
                        print(f"      Units: {balance['units']}")
                        print(f"      Price: ${balance['price']:.2f}")
                        print(f"      Value: ${balance['value']:.2f}")
        finally:
            await client.close()

    asyncio.run(view_portfolio())
    ```
  </Tab>
</Tabs>

## Get Trading Rules

Before placing orders, fetch the trading rules for your intended trading pair:

<Tabs>
  <Tab title="curl">
    ```bash
    curl -u admin:admin -X 'GET' \
      'http://localhost:8000/connectors/hyperliquid/trading-rules?trading_pairs=HYPE-USDC' \
      -H 'accept: application/json'
    ```

    **Response:**
    ```json
    {
      "HYPE-USDC": {
        "min_order_size": 0,
        "max_order_size": 1e+56,
        "min_price_increment": 0.0001,
        "min_base_amount_increment": 0.01,
        "supports_limit_orders": true,
        "supports_market_orders": true,
        "buy_order_collateral_token": "USDC",
        "sell_order_collateral_token": "USDC"
      }
    }
    ```
  </Tab>
  <Tab title="Python">
    ```python
    async def get_trading_rules():
        await client.init()
        try:
            rules = await client.connectors.get_trading_rules(
                connector_name="hyperliquid",
                trading_pairs=["HYPE-USDC"]
            )
            hype_rules = rules["HYPE-USDC"]
            print("Trading Rules for HYPE-USDC:")
            print(f"  Min Order Size: {hype_rules['min_order_size']}")
            print(f"  Min Price Increment: {hype_rules['min_price_increment']}")
            print(f"  Min Base Amount Increment: {hype_rules['min_base_amount_increment']}")
            print(f"  Supports Limit Orders: {hype_rules['supports_limit_orders']}")
            print(f"  Supports Market Orders: {hype_rules['supports_market_orders']}")
            return rules
        finally:
            await client.close()

    asyncio.run(get_trading_rules())
    ```
  </Tab>
</Tabs>

## Place a Limit Order

Execute a limit sell order:

<Tabs>
  <Tab title="curl">
    ```bash
    curl -u admin:admin -X 'POST' \
      'http://localhost:8000/trading/orders' \
      -H 'accept: application/json' \
      -H 'Content-Type: application/json' \
      -d '{
      "account_name": "master_account",
      "connector_name": "hyperliquid",
      "trading_pair": "HYPE-USDC",
      "trade_type": "SELL",
      "amount": 1,
      "order_type": "LIMIT",
      "price": 47.1,
      "position_action": "OPEN"
    }'
    ```
  </Tab>
  <Tab title="Python">
    ```python
    async def place_limit_order():
        await client.init()
        try:
            order = await client.trading.place_order(
                account_name="master_account",
                connector_name="hyperliquid",
                trading_pair="HYPE-USDC",
                trade_type="SELL",
                amount=1,
                order_type="LIMIT",
                price=47.1,
                position_action="OPEN"
            )
            print(f"\nOrder Placed Successfully!")
            print(f"  Order ID: {order['order_id']}")
            print(f"  Status: {order['status']}")
            print(f"  Type: {order['order_type']} {order['trade_type']}")
            print(f"  Amount: {order['amount']} HYPE")
            print(f"  Price: ${order['price']}")
            return order
        finally:
            await client.close()

    asyncio.run(place_limit_order())
    ```
  </Tab>
</Tabs>

<Warning>
If you receive an error like `"Failed to place trade: No order book exists for 'HYPE-USDC'"`, this may indicate you are geo-restricted from trading on the exchange. Check your API logs:
```bash
docker logs hummingbot-api
```
</Warning>

## Complete Example

Here's a complete example that performs all operations:

```python
import asyncio
from hummingbot_api_client import HummingbotAPIClient

async def main():
    client = HummingbotAPIClient(
        base_url="http://localhost:8000",
        username="admin",
        password="admin"
    )
    await client.init()

    try:
        # Step 1: Add credentials
        print("Adding Hyperliquid Credentials...")
        credentials = {
            "hyperliquid_api_key": "0x1234...abcd",
            "hyperliquid_api_secret": "your-private-key",
            "use_vault": False
        }
        await client.accounts.add_credential(
            "master_account", "hyperliquid", credentials
        )

        # Wait for account sync
        await asyncio.sleep(2)

        # Step 2: View portfolio
        print("\nFetching Portfolio...")
        portfolio = await client.portfolio.get_state()
        for account, exchanges in portfolio.items():
            for exchange, balances in exchanges.items():
                for balance in balances:
                    print(f"  {account}/{exchange}: {balance['units']:.2f} {balance['token']}")

        # Step 3: Get trading rules
        print("\nGetting Trading Rules...")
        rules = await client.connectors.get_trading_rules(
            connector_name="hyperliquid",
            trading_pairs=["HYPE-USDC"]
        )
        print(f"  Min Price Increment: {rules['HYPE-USDC']['min_price_increment']}")

        # Step 4: Place order
        print("\nPlacing Limit Order...")
        order = await client.trading.place_order(
            account_name="master_account",
            connector_name="hyperliquid",
            trading_pair="HYPE-USDC",
            trade_type="SELL",
            amount=1,
            order_type="LIMIT",
            price=47.1,
            position_action="OPEN"
        )
        print(f"  Order {order['order_id']} submitted")

    except Exception as e:
        print(f"Error: {e}")
    finally:
        await client.close()

if __name__ == "__main__":
    asyncio.run(main())
```
